## 主机配置：

显卡：AMD Radeon RX470  免驱动

声卡：Realtek ALC 1220-VB   输出正常，麦克风不工作

内存：芝奇幻光戟3000

无线网卡蓝牙：BCM943602CS   原生免驱，需要购买。替换自带无线网卡即可
